# Design OS

AI 기반 브랜드·제품·프로젝트 운영 자동화 시스템입니다.

Design OS는 단순한 디자인 프롬프트 모음이 아니라, 브랜드 데이터와 프로젝트 기록을 축적하여 상세페이지, 카드뉴스, 블로그 이미지, 광고 소재, SNS 콘텐츠, 고객 분석, 경쟁사 분석, 성과 학습까지 연결하는 운영 시스템입니다.

---

## 핵심 목표

- Brand Archive 구축
- Product Archive 구축
- Project Archive 구축
- Customer Voice Archive 구축
- Competitor Archive 구축

- AI Design Team 자동화 기반 마련

## 운영 원칙

디자인은 목적이 아니라 브랜드 목표를 달성하기 위한 수단입니다.

모든 디자인은 다음 요소를 기반으로 설계합니다.

- 브랜드 전략
- 고객 심리
- 제품 USP
- 플랫폼 특성
- 경쟁사 분석
- 고객 언어
- 성과 데이터
- 원본 자산 보호

---

## 로드맵

### Phase 1. Archive 구축

브랜드, 제품, 프로젝트, 고객 언어, 경쟁사, 프롬프트, 성공 사례를 문서화합니다.

### Phase 2. MVP 개발

GitHub, Vercel, Supabase 기반으로 무료 또는 저비용 웹 대시보드를 구축합니다.

### Phase 3. Auto Research Engine

브랜드명, 제품명, 링크를 입력하면 AI가 브랜드·경쟁사·고객 언어·기회 요소를 수집하고 정리하는 구조를 만듭니다.

### Phase 4. MCP Integration

ChatGPT와 GitHub, Supabase, Google Drive, Design OS DB를 연결합니다.

### Phase 5. AI Design Team

전략가, 카피라이터, 디자이너, QA 담당자, 성과 분석가 역할을 분리하여 자동화합니다.

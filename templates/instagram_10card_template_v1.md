# Instagram 10 Card Template v1

인스타그램 카드뉴스 10장 제작을 위한 표준 템플릿입니다.

## 기본 규격

- 사이즈: 1080 x 1350px
- 비율: 4:5
- 안전영역: 상단 100px, 하단 100px, 좌우 80px
- 스타일: 브랜드별로 지정

## 카드별 역할

| 카드 | 역할 | 목적 |
|---|---|---|
| 01 | Hook | 스크롤 정지 |
| 02 | Empathy | 공감 유도 |
| 03 | Problem | 문제 확대 |
| 04 | Solution | 해결책 등장 |
| 05 | Main USP | 핵심 장점 전달 |
| 06 | Sub USP | 보조 장점 전달 |
| 07 | Use Case | 사용 상황 확장 |
| 08 | Lifestyle | 생활 속 장면 제안 |
| 09 | Comparison | 구매 정당화 |
| 10 | CTA | 저장/구매/브랜드 기억 |

## 공통 레이아웃

### 카드 1

- 상단: 헤드라인
- 중앙: 제품 히어로컷
- 하단: 보조 카피

### 카드 2~8

- 상단: 메시지
- 중앙: 라이프스타일 또는 제품 비주얼
- 하단: 설명 또는 보조 정보

### 카드 9

- 비교표 또는 요약 인포그래픽

### 카드 10

- 브랜드 엔딩
- CTA

## 텍스트 규칙

- 한 카드에 핵심 문장 1개
- 본문은 1~2줄 이내
- 수치는 검증된 것만 사용
- 모바일에서 바로 읽히는 크기 사용

## 이미지 규칙

- 제품은 원본 이미지를 사용
- 제품 색상, 로고, 형태, 비율 변경 금지
- 배경과 그래픽은 브랜드 스타일에 맞춰 구성

## 저장 규칙

각 프로젝트는 아래 파일을 남깁니다.

```text
project_brief_v1.md
art_direction_sheet_v1.md
card_prompts_v1.md
figma_layout_guide_v1.md
qa_report_v1.md
```
